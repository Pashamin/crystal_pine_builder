# CrystalOS

ВНИМАНИЕ! ПРОШИВКА ПОКА РАБОТАЕТ ТОЛЬКО НА УСТРОЙСТВАХ Redmi 7a/8/8a

Версия android (на которой работает прошивка) - 11 (lineageOS 18.1)
