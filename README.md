# Crystal OS

Кастомная прошивка, основанная на **LineageOS** и **Android**, цель которой — оживить старые устройства (максимальная оптимизация), а также улучшить безопасность и стабильность системы.

---

## Поддерживаемые устройства
* **Redmi 7A / 8 / 8A** (устройства на платформе `mi439`)
* *Новые модели скоро...*

> **Версия Android:** 11 (на базе LineageOS 18.1)
